5. РОЗГОРТАННЯ ПРОГРАМНОЇ СИСТЕМИ ТА ДЕМОНСТРАЦІЯ ЇЇ РОБОТИ
5.1 Мета: розгорнути програмну систему та продемонструвати її роботу.

5.2 Хід роботи:
5.2.1. Розгорнути реалізовану програмну систему. 
Інфраструктурні технології системи:
1. Docker та Docker Compose - для контейнеризації та оркестрації сервісів
2. MQTT (Mosquitto) - брокер повідомлень для IoT комунікації
3. Prometheus - система моніторингу метрик
4. PostgreSQL - реляційна база даних
5. ELK Stack - для логування та аналітики
6. AWS як цільова платформа розгортання
Процес розгортання системи:
1. Попередні вимоги:
- Встановлений Docker та Docker Compose
- Налаштовані змінні середовища (.env файл)
- Доступ до AWS (якщо розгортання виконується в хмарі)
2. Кроки розгортання:
а) Клонування репозиторію
б) Налаштування змінних середовища:
- Скопіювати .env.example в .env
- Налаштувати необхідні змінні (DB_HOST, JWT_SECRET, тощо)
в) Запуск системи:
	docker-compose up -d
Це розгорне наступні сервіси:
- API (NestJS backend)
- PostgreSQL база даних
- Mosquitto MQTT брокер
- Prometheus для моніторингу
- ELK stack для аналітики

5.2.2. Перевірити та налаштувати роботу розгорнутої програмної системи.
Перевірка та налаштування:
а) Перевірка статусу сервісів:
	docker-compose ps
б) Перевірка логів:
	docker-compose logs -f [service-name]
в) Перевірка підключення до бази даних:
- База даних має автоматично ініціалізуватися через TypeORM
- Перевірити статус міграцій
г) Перевірка MQTT підключення:
- Mosquitto має прослуховувати порт 1883
- Перевірити підключення IoT пристроїв
д) Перевірка моніторингу:
- Доступ до Prometheus на порту 9090
- Перевірка збору метрик
4. Особливості налаштування IoT пристроїв:
- Налаштування MQTT конфігурації для електронних замків
- Перевірка передачі статусів замків
- Налаштування конфігурації 3D-принтерів
5. Перевірка безпеки:
- Перевірка JWT автентифікації
- Перевірка RBAC (системи ролей)
- Перевірка платіжної системи
6. Моніторинг:
- Налаштування алертів у Prometheus
- Перевірка логування в ELK Stack
- Моніторинг статусу вендингових машин

5.3 Висновки: під час лабораторної роботи розгорнули програмну систему та продемонстрували її роботу.
